Project Description

Fantasy Cricket Game

This is a Fantasy cricket Desktop application build in python that allows user to build their own dream cricket team. It is a cricket game where you create a virtual team of real cricket players and score points depending on how your chosen players perform in matches. This project has been created for adding more fun and entertainment to cricket. The game contains many features which the user finds it very much interesting while accessing it. The interface of this game is very much user friendly and anyone can enjoy playing this game.
